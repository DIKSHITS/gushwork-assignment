# Gushwork Web Developer Assignment

## 🚀 Project Overview
This project is a responsive web page built using **HTML, CSS, and JavaScript (Vanilla)** based on the provided Figma design.

## ✨ Features Implemented
- Sticky Header with scroll behavior
- Hero section with image slider
- Manufacturing Process Tabs (interactive)
- Technical Specifications Table
- Feature Cards Section
- Applications Carousel
- Testimonials Section
- Portfolio Section
- FAQ Accordion
- Resources & Downloads
- Contact CTA Form
- Fully Responsive Design

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla)

## 📱 Responsiveness
- Desktop ✅
- Tablet ✅
- Mobile ✅

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in browser
